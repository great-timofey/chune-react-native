#### installation:
1.  git clone __our-repo-name__
2.  cd __cloned-folder-path__
3.  open terminal
4.  *yarn* / *npm install*
4.1 (IOS only) react-native run-ios
4.2 (Android only) android studio -> open __cloned-folder-path/android__ -> sync project -> run / build
